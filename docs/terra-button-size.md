### Button Size

Buttons can be sized by using the `size` prop on the `<Button />` component.

### Usage

```js
import Button from 'terra-button';

<Button size="huge" />
```

### Prop Details

| Prop   | Type   | Default | Description |
|--------|--------|---------|-------------|
| `size` | String |         | Sets the button size. One of `tiny`, `small`, `medium`, `large`, `huge`. |
