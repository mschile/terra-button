### Button Text

Buttons text can be set by using the `text` prop on the `<Button />` component.

### Usage

```js
import Button from 'terra-button';

<Button text="Hello World" />
```

### Prop Details

| Prop   | Type   | Default | Description           |
|--------|--------|---------|-----------------------|
| `text` | String |         | Sets the button text. |
